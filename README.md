# 🎬 NetflixIQ: Content Performance & Engagement Insight Agent  

An AI-Powered Business Intelligence System using Microsoft Power BI

---

## 🌟 Project Overview  

NetflixIQ is an intelligent analytics system designed to transform Netflix dataset insights into interactive and conversational analysis. By integrating Microsoft Power BI with Copilot Studio, this project enables users to explore content performance, trends, and user engagement using natural language queries.  

Focus: Data Analytics & AI Integration  

---

## 🚀 Key Features  

- Conversational BI: Ask questions like *"Which genre has the highest watch time?"*  
- Interactive Dashboards: Multi-page Power BI reports for deep insights  
- Content Performance Analysis: Identify top-performing movies and shows  
- Trend Analysis: Analyze growth of views and content over time  
- Language Insights: Understand global audience preferences  
- AI Integration: Copilot Agent provides instant business insights  
- Data Cleaning & Transformation: Structured and validated dataset for accurate analysis  

---

## 🛠️ Tech Stack  

- Analytics: Microsoft Power BI (DAX, Power Query)  
- Artificial Intelligence: Microsoft Copilot Studio  
- Data Processing: Microsoft Excel  
- Data Source: Netflix Dataset (Kaggle)  
- Design: Dark Theme Dashboard with modern UI  

---

## 🏗️ System Architecture  

- Data Ingestion: Netflix dataset with multiple attributes (title, genre, views, etc.)  
- Data Processing: Cleaning, formatting, and feature engineering in Excel  
- Analysis Layer: Measures and calculations using DAX in Power BI  
- Visualization Layer: Interactive charts, KPIs, and maps  
- AI Layer: Copilot Agent integrated with dataset for conversational insights  
- Output: Multi-page dashboard with AI-assisted analysis  

---

## 📊 Dashboard Structure  

- **Executive Overview:** KPIs like Total Views, Watch Time, Content Count  
- **Revenue & Engagement Analysis:** Watch time, popularity, and trend analysis  
- **Content & Audience Insights:** Genre, language, and distribution insights  

---

## 📈 Key Insights  

- Movies dominate the platform compared to TV shows  
- Drama and Action genres have highest engagement  
- Views have significantly increased over time  
- English content leads globally, followed by regional languages  
- Higher popularity score correlates with higher watch time  

---

## 🤖 AI Agent Features  

- Multi-level conversational flow  
- Answers queries on content, genre, trends, and engagement  
- Provides business insights in natural language  
- Enhances user interaction with dashboard
